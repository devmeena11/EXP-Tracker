# EXP Tracker Documentation

This folder documents the `src` side of the EXP Tracker Vite app.

Files in this folder:

- `architecture-overview.md`: routing, state flow, and app structure
- `pages-and-components.md`: UI responsibilities and interactions
- `utilities-and-analytics.md`: utility modules and chart/heatmap data flow
- `storage-and-styling.md`: local storage behavior and styling system
- `maintenance-reference.md`: extension notes, chart modes, and common gotchas
